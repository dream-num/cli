# Create a Univer plugin from a template

## Usage

```sh
# npm
npm create @univerjs/cli init <project-name>

# pnpm
pnpm create @univerjs/cli init <project-name>
```

## Development

```sh
pnpm i

npm run dev
```

test, create `filter` plugin

```sh
npm run start init ./packages/sheets-plugin-filter
```

## Release

```sh
npm run release
```
